String : 

string is an sequence  of characters / collection of characters 
also it is an built in class which is present in java.lang 
it is an Non-Primitive datatypes because its contains built-in methods such as(trim(), concat(),etc..) String is an Case Sensitive 

we can create string by two ways : 

1> by using String Literal 
2> by using Object Creation 



1> by using String Literal :  SCP(String Constant Pool)

String name = "rohan"; 


2> by using Object Creation :   heap memory

 String name = new String("rohan");
